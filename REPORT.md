# CIA-1 Report Notes

## Title

Design and Development of an Interactive Web-Based Learning System for Linear Regression

## System Design

The application follows a pipeline-style dashboard with eight main stages:

1. Dataset input
2. Preprocessing
3. Exploratory data analysis
4. Linear regression learning module
5. Training configuration
6. Model training and visualization
7. Prediction and inference
8. Evaluation metrics

The system is fully client-side and implemented using HTML, CSS, and JavaScript. This keeps deployment simple and makes the project easy to demonstrate on any lab machine.

## Explanation Strategy

- Every stage includes a "Why this step?" explanation
- Formulas are shown explicitly
- Before-vs-after comparisons are displayed for preprocessing
- A worked sample explains how one prediction is calculated
- Cost convergence and predicted-vs-actual graphs connect the math to behavior
