# Linear Regression Learning Studio

This project is now a Python-based `Streamlit` machine learning application for understanding linear regression step by step.

## Features

- CSV upload with dataset preview, shape, missing-value count, and data types
- Preprocessing controls for missing values, encoding, scaling, and feature selection
- EDA with histogram, scatter plot, and correlation analysis
- Manual gradient descent for linear regression
- Worked-sample explanation with formulas and intermediate values
- Train-test split and k-fold cross-validation
- Prediction module with step-by-step weighted contribution breakdown
- Evaluation metrics: MSE, MAE, and R²

## Files

- `app.py`: main Streamlit app
- `requirements.txt`: Python dependencies
- `REPORT.md`: report notes for submission

## How To Run

1. Open a terminal in `C:\Users\SANDIP\OneDrive\Documents\New project`

