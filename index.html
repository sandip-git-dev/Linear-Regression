    Educational Notes

    Missing values

    Empty cells distort averages and gradients. Imputation keeps more data while preventing blank values from breaking the math.

    Encoding

    Regression needs numbers. Encoding converts category names into usable numeric inputs while preserving information.

    Scaling

    Scaling brings features to comparable ranges, which helps gradient descent converge more smoothly.

    Why this step?

    If a feature barely changes, it may contribute little. If a feature moves with the target, regression can often capture that trend with coefficients.
